# Star Words - Vocabulary Adventure

Welcome to **Star Words**, an interactive space adventure game designed to help children practice English vocabulary!

## Features
- **Space Theme**: Catch falling "Word Stars" with your rocket ship.
- **Pronunciation**: Hear the word spoken out loud when you catch the correct star.
- **Vocabulary Categories**: Animals, Fruits, Colors, Numbers.
- **Interactive**: Control with Keyboard (Left/Right), Mouse, or Touch.

## How to Run
1. Navigate to this directory in your terminal.
2. Run `npm install` (if you haven't already).
3. Run `npm run dev`.
4. Open the link in your browser.

## Gameplay
- The game will show you a Chinese word (e.g., "è²“") and pronounce the English word ("Cat").
- Your goal is to move your rocket ship to catch the falling star that matches the English word.
- Avoid the wrong words!
- Have fun and learn!
