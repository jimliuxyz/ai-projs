---
description: 建立新的單字組 (Create a new vocabulary word set)
---

1. 詢問使用者想要建立的單字列表或主題。
2. 詢問偏好的檔案名稱（例如：`20260114` 或 `fruit`）。若使用者未指定，建議使用當前日期 `YYYYMMDD` 格式。
3. 針對每個單字，透過以下結構生成高品質的內容：
   - `q`: 英文單字或短語。
   - `t`: 精確的繁體中文翻譯。
   - `tags`: 分類標籤陣列（例如：`["verb"]`, `["noun"]`, `["daily"]`, `["nature"]`）。
   - `exps`: 1-3 個精心雕琢的例句。
     - **雕琢準則**：
       - **具備情境感**：避免如 "I like Apple" 等過於直白或缺乏深度的句子。
       - **實用性**：優先選擇日常生活中自然出現的慣用語或對話片段。
       - **關聯性**：例句應能清晰展示該單字在特定語境下的正確用法與搭配詞。
       - **長度適中**：適合學習與朗讀，不宜過於冗長。
4. 將生成的 JSON 內容預覽給使用者確認，並特別指出例句的優點。
5. 在使用者確認後，將 JSON 檔案儲存至 `src/assets/vocabularies/{filename}.json`。
6. 通知使用者單字組已建立完成。
